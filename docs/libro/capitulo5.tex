\section{Discusión}

\subsection{Hallazgos Principales}

\begin{itemize}
    \item El framework propuesto mejora significativamente las métricas
    \item Múltiples métodos MCDM convergen a soluciones similares
    \item Trade-offs entre métricas son manejados efectivamente
    \item La validación por expertos confirma la calidad visual
\end{itemize}

\subsection{Ventajas del Enfoque}

\begin{itemize}
    \item Automatización completa del proceso
    \item Exploración exhaustiva del espacio de parámetros
    \item Robustez mediante consenso MCDM
    \item Flexibilidad para incorporar preferencias
\end{itemize}

\subsection{Limitaciones}

\begin{itemize}
    \item Tiempo computacional relativamente alto
    \item Necesidad de validación con datasets más grandes
    \item Dependencia de la calidad de la imagen original
    \item Posibles artefactos en regiones de muy bajo contraste
\end{itemize}
